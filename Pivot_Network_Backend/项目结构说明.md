# 项目结构说明

更新时间：`2026-04-12`

## 1. 文档目的

这份文档只回答一个问题：

- 当前仓库里哪些目录是正式活动目录，哪些只是历史归档或参考材料

## 2. 当前正式活动目录

### 2.1 `Seller_Client/`

当前正式卖家客户端目录。

### 2.2 `Buyer_Client/`

当前正式买家客户端目录。

### 2.3 `Plantform_Backend/`

当前正式平台后端目录。

### 2.4 `Docker_Swarm/`

当前正式 Docker Swarm 基础设施目录。

### 2.5 `wireguard/`

当前正式 WireGuard 基础设施与部署资产目录。

### 2.6 `docs/`

当前项目文档、教程与 runbook 目录。

## 3. 当前辅助但仍保留的目录

### 3.1 `archive/`

当前项目自己导出的历史环境快照与迁移前状态。

### 3.2 `_client_backups/`

客户端阶段性备份包。

### 3.3 `env_setup_and_install/`

环境准备与 Codex / CCCC 配置模板。

### 3.4 `win_romote/`

Windows 远程 SSH 与辅助脚本说明。

### 3.5 `Shared/`

存在，但不是 seller / buyer / backend / swarm 的主开发入口。

## 4. 当前历史参考目录

### 4.1 `backup_wasted/`

历史实现与废弃参考材料目录，不是当前主开发目录。

## 5. 当前建议目录树

```text
/root/Pivot_network
├── Seller_Client/                    # 当前正式卖家客户端
├── Buyer_Client/                     # 当前正式买家客户端
├── Plantform_Backend/                # 当前正式平台后端
├── Docker_Swarm/                     # 当前正式 Swarm 基础设施
│   └── Docker_Swarm_Adapter/         # 当前正式私有基础设施控制面
├── wireguard/                        # 当前正式 WireGuard 资产
├── docs/                             # 当前文档 / 教程 / runbooks
├── archive/                          # 历史快照
├── _client_backups/                  # 客户端备份包
├── env_setup_and_install/            # 环境配置模板
├── win_romote/                       # Windows 远程辅助说明
├── backup_wasted/                    # 历史实现与废弃参考
├── PROJECT.md
├── 架构说明.md
├── 项目结构说明.md
└── 个人算力交易平台MVP技术方案.md
```

## 6. 当前开发约束

- 新功能优先写入 `Seller_Client/`、`Buyer_Client/`、`Plantform_Backend/`、`Docker_Swarm/`
- 不把 `backup_wasted/` 当主开发目录
- 顶层说明以当前代码和当前目录现实为准，不以历史规划图为准
