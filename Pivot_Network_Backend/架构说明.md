## 当前总体架构

当前实际架构已经落成并验证为两条相连的链：

1. seller 接入与商品化链
   - `Seller_Client -> Plantform_Backend -> Docker_Swarm_Adapter -> Docker Swarm`
2. buyer 交易与会话链
   - `Buyer_Client -> Plantform_Backend -> Docker_Swarm_Adapter / WireGuard / RuntimeSession`

## 当前最重要的系统语义

- seller 卖的是可售算力节点
- offer 是平台对 seller 节点的商品化表达
- buyer 买的是 `RuntimeSession`
- buyer 在浏览器里操作的是 `RuntimeSession` 的 shell，而不是 seller 宿主机本身
